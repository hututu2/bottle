是不需要自己涂的，点一点就能填好的小瓶子～
（小瓶子的图片已附带）

这个网页大概分成以下几个部分（可以参考那个录屏）：
1. 设定标题（比如“心动小瓶子”）
2. 选择添加 / 删除一个小瓶子
3. 点击一下小瓶子就可以把它填满一些～（如果已经满了就变回空瓶，循环）

一些要求和提示：
1. 要求使用 JS 生成小瓶子的 HTML 元素然后插入页面（你可以先写好 HTML，再想想怎样可以用 JS 做成这个样子）
2. 小瓶子当然是用 <img src="..."> 实现了，在 onclick 事件中修改 src，小瓶子的样子可以使用一个整数记录在变量 / <img> 的自定义属性里～
3. 自定义的属性一般以 data- 开头，例如 data-level。设置属性：***.setAttribute('data-level', ***) 获取属性：***.getAttribute('data-level')，注意：获取属性的值得到的是字符串，别忘了 parseInt
4. 为了方便布局，我把 bootstrap 的“十二列网格布局”抽了出来，放在 bootstrap-grid-system.css 这个文件里，引入这个 CSS 之后适当地添加 class="col-*-*" 就可以实现某个 <div> 占一行的长度的十二分之几～具体请参见 https://www.runoob.com/bootstrap4/bootstrap4-grid-basic.html
5. 如果你愿意的话，还可以试着用“十二列网格布局”适配一下手机版的界面样式|•'-'•) ✧
6. 新建元素：document.createElement 插入元素：***.appendChild ***.insertBefore 删除元素：***.removeChild
7. 不要使用 jQuery 等第三方库，不要使用 Vue 之类的 MVVM 框架（如果你知道这东西是什么以及怎么用的话……虽然用这个写确实会非常舒服）

9. 小瓶子的图片和 CSS 与网页放在同一路径下并且不要改名，交作业只需要交 HTML 文件
